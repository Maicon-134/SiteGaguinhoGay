# HiagoXiter STORE

Site **hiagoxiter.com** (React + Vite).

## Editar o site

Trabalhe na branch **`develop`**:

```bash
git checkout develop
npm install
npm run dev
```

## Publicar na Hostinger

A branch **`main`** contĂŠm o site compilado. A Hostinger deve usar:

- Branch: **`main`**
- DiretĂłrio: **`public_html`**

Cada push na `develop` atualiza a `main` automaticamente via GitHub Actions.

Guia completo: [HOSTINGER-GIT.md](HOSTINGER-GIT.md)
